# GalAssetBox Plugin Package Demo

This folder demonstrates the shareable plugin package shape.

Files:

- `plugin.json`: package manifest
- `plugin.gabplugin.js`: plugin entry script
- `sample.packagepack.json`: tiny test data

The plugin is only a safe demo for creator-owned or explicitly authorized data. It does not include third-party game rules, keys, cracking logic, or DRM bypass behavior.
